# previous_assignments — your style/tone reference

Drop your **last ~5 assignments** here. Any format — essay (PDF/DOCX), PowerPoint (PPTX), or other.
This is the reference set the team uses to keep the new work sounding like *you*.

- **Alex** (authenticity & human voice) compares the draft against these in every revision/FQA loop,
  flagging passages that deviate from your established style or read as AI-generated.
- **Pascal** (tone) builds a style profile from them — sentence length, vocabulary, voice, favoured
  transitions — and steers drafting/revision to match.

Optional: keep a `student-style-profile.md` here (Pascal/Alex maintain it) summarising the extracted
patterns so agents don't re-derive them every run.

Prefer the **most recent** assignments — if older ones conflict in style, the newest win.
