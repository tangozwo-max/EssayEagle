# 00 Input — Assignment intake

Drop everything the assignment gives you here. This is **not** a workflow; it feeds the
Preparation phase (Setup workflow).

Expected files (names matter — the app's `startup` route looks for them):

- `assessment_brief.md` — the assignment brief
- `grading_rubric.md` — the marking rubric
- `referencing_style_guide.md` — referencing style (e.g. Harvard-WMS)
- `curriculum/` — module / curriculum materials (add a `README.md` index)
- `previous_assignments/` — prior submissions for reference (add an `index.md`)

Tip: a brief/rubric/style file may contain a fenced code block with the absolute source path
of the original document; the app reads that to link back to the source.
