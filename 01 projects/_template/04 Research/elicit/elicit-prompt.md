## How to run
1. Open **elicit.com** → "Find papers".
2. Paste each **Research question** below, one at a time.
3. Add the **Columns** listed (Elicit → "Add column" → custom).
4. Apply the **Filters** (study type / years / population).
5. Export each result (PDF or CSV) → save into `04 Research/elicit/report/`.
6. Tell the orchestrator to continue (`/fabrik-run <project>`), and Jackie ingests it.
