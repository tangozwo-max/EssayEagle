# M5 W2 P16 — The Addiction Cycle

Introduces the three-stage addiction cycle model (binge/intoxication → withdrawal/negative affect → preoccupation/anticipation) based on animal studies by Koob et al., confirmed in humans by Volkow. Each stage maps to a specific brain region. DSM-5 SUD diagnostic criteria are reviewed across four categories (impaired control, social problems, risky use, physical dependence). The concept of allostasis is introduced as the brain's feed-forward adaptive mechanism that shifts the reward set-point. Anti-reward systems (CRF, norepinephrine, dynorphin) are described as emerging during deep addiction. Key behaviours in the cycle include impulsivity, positive reinforcement, negative reinforcement, and compulsivity.
