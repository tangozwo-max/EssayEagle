# M4 W1 P03 — Still face experiment

This page describes Dr. Edward Tronick's Still Face Experiment (1978), a landmark paradigm demonstrating infant distress in response to caregiver unresponsiveness and the critical importance of consistent, responsive caregiving for infant emotional regulation and attachment security.
