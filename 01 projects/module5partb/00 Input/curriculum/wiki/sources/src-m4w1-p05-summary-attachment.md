# M4 W1 P05 — Summary for Attachment

Key learning points for the attachment section. Covers Bowlby, secure vs insecure attachment, the Still Face Experiment, and how attachment styles manifest in healthcare interactions.

## Key concepts covered
- [[attachment-theory]] - Bowlby — lasting psychological connectedness
- [[secure-attachment]] - safe haven, proximity maintenance, secure base, separation anxiety
- [[insecure-attachment]] - maladaptive patterns from inconsistent caregiving
- [[still-face-experiment]] - Tronick 1978 — infant distress with unresponsive caregiver
- [[attachment-styles]] - secure, anxious, avoidant, disorganised
- [[internal-working-model]] - shapes interactions in healthcare settings
