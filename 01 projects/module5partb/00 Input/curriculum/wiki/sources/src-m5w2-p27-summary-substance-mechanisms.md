# M5 W2 P27 — Summary: Substance Mechanisms of Action

End-of-section summary for P22–P26 covering mechanisms of action for the main substances in M5 W2. Consolidates key learning points across alcohol, cannabis, benzodiazepines, cocaine, and methamphetamine.

## Key learning points
- AUD is a brain disorder; alcohol misuse during adolescence causes lasting structural changes
- Cannabis alters self-awareness networks; reduces anxiety, impairs memory, affects motivation
- BZD are life-threatening in combination with opioids (CNS depression); abrupt withdrawal can cause fatal convulsions
- Cocaine causes dopamine excess via DAT blockade; also drives cortisol overproduction
- Methamphetamines are highly neurotoxic; cause neuronal inflammation and degeneration via glutamate
