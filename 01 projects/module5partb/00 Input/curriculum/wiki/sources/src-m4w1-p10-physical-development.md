# M4 W1 P10 — Physical Development

Overview of physical development milestones across infancy, toddlerhood, pre-school, childhood, and adolescence. Covers motor skill progression (gross and fine), reflexes in newborns, and puberty-related changes.

## Key concepts covered
- [[gross-motor-skills]] — large body movements (walking, jumping, climbing)
- [[fine-motor-skills]] — hand-eye coordination, drawing, cutting
- [[grasp-reflex]] — newborn reflex closing fingers when palm is stroked
- [[rooting-reflex]] — newborn reflex aiding feeding
- [[developmental-milestones]] — age-linked physical benchmarks across childhood
- [[puberty]] — hormonal changes leading to secondary sexual characteristics
