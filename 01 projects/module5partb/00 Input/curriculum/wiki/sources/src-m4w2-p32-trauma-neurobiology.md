# M4 W2 P32 — Trauma and neurobiological development

Describes how childhood abuse and neglect alter brain development through changes to threat, reward, and memory systems. Introduces "latent vulnerability" concept: adaptive changes that help children cope in harmful environments but create difficulties in safer settings. Details impacts on hippocampal memory processing, amygdala threat detection, and nucleus accumbens reward processing.

## Key concepts covered
- [[latent-vulnerability]] — increased future mental health risk from early trauma
- [[threat-system]] — amygdala-based danger detection
- [[memory-system]] — hippocampal autobiographical memory processing
- [[reward-system]] — nucleus accumbens pleasure and motivation
- [[hypervigilance]] — chronic threat detection
- [[anhedonia]] — inability to feel pleasure
