# M4 W1 P09 — Developmental milestones

This page introduces developmental milestones as age-specific skills and behaviours used to track child progress and identify delays. It describes five interconnected developmental domains (physical, cognitive, language/communication, social, emotional), principles of development, and considerations for using milestones clinically. The page emphasises cultural and neurodevelopmental variation in milestone attainment.
