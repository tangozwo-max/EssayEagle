# Mark Griffiths

One of the world's foremost experts in behavioral addiction research. Griffiths developed — building on R.I. Brown's earlier conceptual work — the six-criteria diagnostic framework for behavioral addiction that has become standard in the field.

## Key contributions
- Developed the [[griffiths-brown-criteria]]: six-criterion framework (salience, euphoria, tolerance, withdrawal, conflict, relapse/reinstatement) for defining behavioral addictions
- Extensive research on gambling, gaming, and internet addiction
- Distinguished between compulsive behavior and behavioral addiction, requiring functional impairment for the latter
