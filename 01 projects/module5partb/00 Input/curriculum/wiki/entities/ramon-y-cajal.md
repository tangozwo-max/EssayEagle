# Santiago Ramón y Cajal (1852–1934)

Spanish histologist / pathologist / neuroscientist. Developed the **neuron theory** using Golgi staining; produced extraordinarily detailed neuron drawings. Shared 1906 Nobel Prize with Camillo Golgi (while publicly disagreeing with him).
