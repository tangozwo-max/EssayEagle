# Adrenaline (Epinephrine)

Hormone exclusively produced by the medulla of the adrenal gland. Synthesised from noradrenaline. Mediates the fight-or-flight response — increases HR, BP, respiration, carbohydrate metabolism; prepares muscles for action.

## Adrenaline vs Noradrenaline

| | **Adrenaline** | **Noradrenaline** |
|---|---|---|
| Source | Adrenal medulla only | Adrenal medulla + sympathetic nerves |
| Function | Hormone | Hormone + neurotransmitter |
| Main effect | Energy mobilisation | ↑ BP via vasoconstriction |
| Synthesised from | Noradrenaline | Dopamine |
