# Social-Cognitive Model of Stigma

Extends [[modified-labeling-theory]]. Public stigma arises through a **cognitive-behavioural process**:

```
Social cue (eccentricity, symptom, label)
     │
     ▼
Stereotype (cognition)
     │
     ▼
Prejudice (emotion)
     │
     ▼
Discrimination (behaviour)
```
