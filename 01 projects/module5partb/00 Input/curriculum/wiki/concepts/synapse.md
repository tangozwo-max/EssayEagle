# Synapse

Specialised junction where communication occurs between two neurons (or between a neuron and another cell type — muscle, gland).

## Components
- **Presynaptic terminal** — end of the sending neuron's axon; contains synaptic vesicles filled with neurotransmitters.
- **Synaptic cleft** — ~40 nm gap.
- **Postsynaptic membrane** — contains neurotransmitter receptors.

## Process
1. Action potential reaches presynaptic terminal → voltage-gated Ca²⁺ channels open.
2. Ca²⁺ influx → synaptic vesicles fuse and release neurotransmitter.
3. NT diffuses across synaptic cleft, binds postsynaptic receptors.
4. Postsynaptic response: ion channels open → excitatory or inhibitory effect.
5. Signal termination via degradation, reuptake, or diffusion.
