# Neuron Structure

~**85 billion neurons** in the human brain. Structure analogous to a tree.

| Component | Function |
|---|---|
| **Soma (cell body)** | Contains nucleus; integrates incoming signals |
| **Cell nucleus** | Controls gene expression and protein synthesis |
| **Dendrites** | Branches carrying receptors that pick up signals from other neurons |
| **Axon** | Long projection carrying electrical impulses *away* from soma |
| **Myelin** | Fatty insulation enabling rapid impulse transmission |
| **Synapse** | Specialised junction for cell-to-cell communication |
| **Axon terminal** | End of axon; releases neurotransmitters |

## Axon ensheathing cells
- PNS → **Schwann cells**
- CNS → **Oligodendrocytes**

Myelin deficits contribute to schizophrenia, depression, addiction, and multiple sclerosis.
