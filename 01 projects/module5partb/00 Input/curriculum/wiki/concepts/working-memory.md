# Working Memory

Working memory is a foundational component of executive function that determines how information is temporarily stored, processed, and manipulated. Functioning as the brain's mental workspace, working memory holds information active in consciousness whilst the person performs cognitive operations upon it—learning, reasoning, comprehension, and problem-solving. The capacity of working memory is limited, and individual differences in working memory capacity have substantial implications for academic achievement, occupational functioning, and quality of life.

## Function and Architecture

Working memory enables individuals to:
- Retain information temporarily (seconds to minutes)
- Manipulate information whilst holding it in mind (e.g., mental arithmetic)
- Use retained information to guide ongoing behaviour
- Suppress irrelevant information while focusing on task-relevant details

The central executive component directs attention and coordinates the phonological loop (verbal-acoustic information) and visuospatial sketchpad (visual-spatial information), enabling complex cognitive operations.

## Neurodevelopmental Impairment

Children with neurodevelopmental conditions often experience working memory difficulties:

- **ADHD**: Reduced working memory capacity and control impair sustained attention and impulse inhibition
- **Specific learning disorders**: Dyslexia and dyscalculia are frequently accompanied by working memory deficits
- **Auditory processing disorder**: Children must allocate substantial working memory resources merely to decode verbal input, leaving reduced capacity for comprehension
- **Developmental coordination disorder (DCD)**: Motor planning and execution demands consume working memory resources

## Educational and Clinical Implications

Children with working memory impairments may appear inattentive, disorganised, or unmotivated when they are actually struggling to process information efficiently. Teachers and mental health professionals who recognise these difficulties can provide concrete support: written instructions, visual aids, breaking tasks into smaller steps, allowing extra processing time, and reducing extraneous cognitive load.
