# PERMA™ — Seligman's Theory of Well-Being

Framework by **Dr. Martin Seligman** (positive psychology) answering: *"What is human flourishing and what enables it?"*

| Letter | Element | Description |
|---|---|---|
| **P** | Positive Emotions | Happiness, joy, gratitude, contentment. Cultivatable via past (gratitude/forgiveness), present (mindfulness), future (hope/optimism). |
| **E** | Engagement | Flow — full immersion in a challenging activity where skills match demand. |
| **R** | Relationships | Positive social connections; acts of kindness boost own well-being. |
| **M** | Meaning | Purpose, values alignment, contribution to something larger than self. |
| **A** | Accomplishments | Setting and achieving goals → competence, mastery, self-efficacy. |

## Real-world benefits of high well-being
Better work performance, satisfying relationships, stronger immunity, better physical health, longer life, reduced CV mortality, fewer sleep problems, less burnout, better self-regulation, more prosocial behaviour.
