# Neurotransmitters

**Chemical signals** that enable communication between one cell and a target cell (next nerve, muscle, or gland). Key regulators of mood, emotions, behaviour, and cognition.

## Key neurotransmitters in mental health

| Neurotransmitter | Role | Dysregulation linked to |
|---|---|---|
| **Serotonin (5-HT)** | Mood, sleep, appetite | Depression, anxiety |
| **Dopamine (DA)** | Reward, motivation, motor control | Schizophrenia (excess), Parkinson's (deficit) |
| **Norepinephrine (NE)** | Arousal, alertness, stress response | Depression, anxiety, PTSD |
| **GABA** | Primary inhibitory NT | Anxiety, epilepsy |
| **Glutamate** | Primary excitatory NT | Schizophrenia, mood disorders |
| **Acetylcholine** | Learning, memory | Alzheimer's disease |
