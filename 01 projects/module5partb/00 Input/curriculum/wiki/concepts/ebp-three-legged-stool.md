# EBP Three-Legged Stool

Visual metaphor for the three equally weighted pillars of [[evidence-based-practice]]. Removing any leg collapses the stool.

```
       [ Clinical decision ]
        /        |         \
   Research   Clinical    Person's
   evidence   expertise   values /
                          preferences
```
