# Retraumatisation

Process by which a trauma survivor's trauma memories are reactivated — by clinical process, environment, or behaviour — intensifying distress. A core concern of trauma-informed care.

## Prevention strategies
- LIVES framework (Listen, Inquire, Validate, Enhance Safety, Support)
- Informed consent before assessment detail
- Grounding techniques if overwhelmed
