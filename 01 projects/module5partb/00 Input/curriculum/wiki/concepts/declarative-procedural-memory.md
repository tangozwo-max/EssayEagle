# Declarative vs Procedural Memory

- **Declarative memory** — recall of facts, life events, personal experiences, recognising family members. Regions: **hippocampus, temporal lobe**.
- **Procedural memory** — riding a bike, driving, playing an instrument. Regions: **cerebellum, basal ganglia**.

People with Alzheimer's disease often lose declarative memory but retain procedural memory — different brain regions show different vulnerability to neurodegeneration.
