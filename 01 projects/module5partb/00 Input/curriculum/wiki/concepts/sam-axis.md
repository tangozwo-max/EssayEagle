# SAM Axis — Sympathetic-Adreno-Medullary

**Acute, rapid stress-response system**. Complementary to the slower [[hpa-axis]].

## Mechanism
Sympathetic nervous system → adrenal medulla → release of **adrenaline** and **noradrenaline** within seconds.

## Comparison with HPA axis

| | **SAM axis** | **HPA axis** |
|---|---|---|
| Function | Rapid fight-or-flight | Long-term adaptation |
| Stress type | Acute (physical, one-off) | Chronic (psychological) |
| Main hormones | Adrenaline, noradrenaline | Cortisol |
| Response time | Seconds | Minutes–hours |
| Duration | Seconds–minutes | Hours–days+ |
| Physiology | ↑ HR, ↑ BP, tunnel vision | ↑ glucose, ↑ appetite, ↓ immunity |
| Plasticity effect | Minimal | Promotes/impairs plasticity |
| Adaptation | Short-term | Long-term — resilience or pathology |
