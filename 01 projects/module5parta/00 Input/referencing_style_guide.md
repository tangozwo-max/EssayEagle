**Key rules (quick reference):**
- In-text: Author (Year) or (Author, Year) — no first names, no "et al." until 4+ authors
- Reference list: alphabetical by surname, hanging indent
- No URLs in running text; full URL + accessed date in reference list
- Page numbers required for direct quotes
